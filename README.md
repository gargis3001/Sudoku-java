# 🧩 Java Sudoku Game

A desktop-based **Sudoku game** built in **Java**. This project allows users to play Sudoku.

---

## 📌 Features

- ✅ 9x9 Sudoku board
- 🧠 Input validation 
- 🎮 Interactive GUI 

---

## 🛠️ Tech Stack

- **Language**: Java  
- **IDE**: IntelliJ IDEA  
- **GUI**: JavaFX  
- **Version Control**: Git & GitHub
